# Wheels output directory

Copy built **OpenCV + GStreamer** wheels from `build_workspace/` here, organized by Python version:

| Directory | Purpose |
|-----------|---------|
| `cp311/` | Python 3.11 wheels |
| `cp312/` | Python 3.12 wheels |
| `cp313/` | Python 3.13 wheels |

Then upload from here to **GitHub Releases** or **PyPI**.
