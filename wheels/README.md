# Wheels 輸出目錄

建置產生的 **OpenCV + GStreamer** wheel 依 **OpenCV 版本** 與 **Python 版本** 分類：

| 目錄 | 說明 |
|------|------|
| `4.9/cp311/` | OpenCV 4.9.x，Python 3.11 |
| `4.9/cp312/` | OpenCV 4.9.x，Python 3.12 |
| `4.9/cp313/` | OpenCV 4.9.x，Python 3.13 |
| `4.13/cp311/` | OpenCV 4.13.x，Python 3.11 |
| `4.13/cp312/` | OpenCV 4.13.x，Python 3.12 |
| `4.13/cp313/` | OpenCV 4.13.x，Python 3.13 |

可由此目錄上傳至 **GitHub Releases** 或 **PyPI**。
